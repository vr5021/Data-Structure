88. Merge Sorted Array
```
Easy

You are given two integer arrays nums1 and nums2, sorted in non-decreasing order,
and two integers m and n, representing the number of elements in nums1 and nums2 respectively.

Merge nums1 and nums2 into a single array sorted in non-decreasing order.

The final sorted array should not be returned by the function, but instead be stored inside the array nums1.      
To accommodate this, nums1 has a length of m + n, where the first m elements denote the elements
that should be merged, and the last n elements are set to 0 and should be ignored. nums2 has a length of n.
```
My submission
```python
class Solution(object):
    def merge(self, nums1, m, nums2, n):
        """
        :type nums1: List[int]
        :type m: int
        :type nums2: List[int]
        :type n: int
        :rtype: None Do not return anything, modify nums1 in-place instead.
        """
        # nums1의 끝 인덱스
        i = m - 1
        # nums2의 끝 인덱스
        j = n - 1
        # 합쳐진 배열의 끝 인덱스
        k = m + n - 1

        # 두 배열의 끝에서부터 비교하면서 큰 값을 nums1의 끝부터 채워나감
        while i >= 0 and j >= 0:
            if nums1[i] > nums2[j]:
                nums1[k] = nums1[i]
                i -= 1
            else:
                nums1[k] = nums2[j]
                j -= 1
            k -= 1

        # nums2에 남은 값들을 nums1의 앞부분에 채워넣음
        nums1[:j+1] = nums2[:j+1]
```
